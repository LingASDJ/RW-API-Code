## &nbsp; One.什么是高级调试Debug?

铁锈战争是一个多语言，有很多元素的一款游戏。  
但我们可爱的开发者Luke,设置里面远远没有包含全部游戏设置。  
在 1.14 开始，卢克偷偷加了一个 Debug 按钮并且不做任何说明与
提示。   
我通过筛查Dex我发现了这个东西，并将资料整理出来。
